# 疲劳检测系统简介 python328

## 项目概述

本项目是基于Python的疲劳检测系统，通过集成opencv库实现功能。系统功能齐全，既可作为项目实训，也可用于在线课程任务。

## 技术栈

- 开发语言：Python 3.6.8
- 数据库：MySQL 5.7
- 数据库管理工具：Navicat11
- 开发工具：PyCharm

## 功能模块

系统支持以下功能模块：

- 人脸识别
- 图像分析
- 疲劳检测

## 系统角色

- 用户：使用系统进行疲劳检测

## 运行环境

- Python 3.6.8
- MySQL 5.7

## 部署步骤

1. 安装Python 3.6.8
2. 安装MySQL 5.7
3. 导入数据库SQL文件
4. 安装所需第三方库
5. 运行程序
